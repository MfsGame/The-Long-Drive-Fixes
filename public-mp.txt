#       because of host wheel-suspension fight; now safe with v0.5 of the
#       kinematic plugin protecting the non-driver's side).

#   - TLDPubRemoteCarKinematic 0.5.0 (UPDATED from 0.1.0)
#       Adds otherClaimed clause to the authority check. When a client
#       claims a car the host is also rendering, host now goes kinematic on
#       that car too, instead of running double-physics that fights the
#       client's authoritative broadcasts.
